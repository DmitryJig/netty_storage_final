Десктопное приложение "Сетевое хранилище"
реализовано на версии Java 1.8 чтобы не возиться с подключением библиотек JavaFx

- модуль сервера server с передачей данных в формате json
    стартовый файл :
    src/main/java/org/example/server/Server.java

- модуль client_json - реализован UI на javaFX с передачей данных в формате json,
        стартовый файл:
    src/main/java/org/example/Main.java

- Назначение кнопок:
    "Update file list in user directory" обновляем список файлов в директории клиента(в т.ч. во вложенных папках)
    "Update file list in server directory" обновляем список файлов в директории на сервере(в т.ч. во вложенных папках)
    "send to server" отправить выделенный файл из директории клиента на сервер
    "delete in client" удалить выделенный файл в директории клиента
    "send to client" отправить выделенный файл с сервера в директорию клиента
    "delete in server" удалить выделенный файл с сервера

выбираем файл для отправки на сервер, нажав кнопку "send to server" отправляем файл на сервер,
так же работает отправка с сервера на клиент
    - Удаляем файл в директории клиента нажав "delete in client", так же удаляем файлы и на сервере

    - работает отправка файлов в обе стороны, удаление файлов, показ списка файлов на сервере и клиенте

    - Пока без аутентификации
